# PM-Quiz-Challenge-Scrum-Master
Project Highlight：AI 驅動的Scrum闖關平台，展現 AI Agentic Workflow 實踐。GenAI Prototyping：運用 Coding Agent 寫入 React/Vite 前端架構，並透過 Prompt Engineering 生成 3D 遊戲資產。 API &amp; Data Flow：利用 Google Apps Script 自行開發 RESTful API，串接 Google Sheets 建立 Serverless 後端與計分系統。 Agentic Workflow：透過 AI 代理人執行 Terminal 指令與檔案管理，從無到有實現自動化部署與產品迭代。
