## 專案簡介 (Project Overview)
這是一個結合 **Scrum/Agile 專案管理實務** 的互動式闖關問答遊戲。本專案展示了如何運用 **AI Agentic Workflow** 進行全端開發，從需求分析、UI 設計到後端資料庫串接，實現高效且現代化的 Web 應用架構。
## 關鍵技術與工具 (Tech Stack & Tools)
本專案展現了跨領域的技術整合能力：
*   **Frontend Engine**: React + Vite (模組化組件設計)
*   **Serverless Backend**: Google Apps Script (GAS) 部署微服務 API
*   **Database**: Google Sheets (低代碼數據管理與視覺化)
*   **AI Integration**:
    *   **Generative AI**: 用於生成「3D 虛擬騎士」關主素材與 UI 程式碼生成。
    *   **Prompt Engineering**: 精準控制法式黑白極簡風格 (French Monochrome Style) 的 CSS 輸出。
## 架構與串聯流程 (Architecture & Workflow)
為了展現 AI 專案落地能力，我建立了以下自動化串聯：
1.  **AI 驅動開發 (AI-Driven Development)**:
    *   透過精確的 Prompt 定義「資深 PM」角色，確保遊戲題目與邏輯符合專業 Scrum 規範。
    *   利用 AI 快速迭代 MVP，將自然語言需求轉化為可執行的 React Hooks 與 State Management 邏輯。
2.  **無伺服器資料流 (Serverless Integration)**:
    *   使用 `App Script` 建立 REST API，解決跨網域 (CORS) 問題。
    *   實現雙向數據流：前端即時撈取題庫 (GET) -> 使用者作答 -> 成績回寫 Sheets (POST)。
3.  **沈浸式體驗設計 (Immersive UX)**:
    *   整合 AI 生成的 3D 藝術資產，提升視覺張力。
    *   實作即時回饋系統，模擬專案闖關的成就感。
## 核心價值 (Core Competencies)
此專案證明了我不僅具備程式開發能力，更擁有：
*   **AI 工具鏈整合能力**：能有效指揮 AI 完成複雜任務。
*   **全端系統架構觀**：低成本、高效率的解決方案設計。
*   **產品思維**：從使用者體驗 (UX) 出發的敏捷開發流程。
